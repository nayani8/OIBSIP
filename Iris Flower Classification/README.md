# 🌸 Iris Flower Classification using Machine Learning

## 📖 Project Overview

This project focuses on classifying Iris flower species using Machine Learning algorithms. The goal is to predict the species of an Iris flower based on its sepal and petal measurements.

The project covers the complete machine learning workflow, including data preprocessing, exploratory data analysis (EDA), model training, evaluation, and comparison.

---

## 🎯 Objective

To build a classification model capable of accurately identifying Iris flower species from flower measurements.

---

## 📊 Dataset Information

The dataset contains the following features:

* Sepal Length (cm)
* Sepal Width (cm)
* Petal Length (cm)
* Petal Width (cm)

Target Variable:

* Species

  * Iris-setosa
  * Iris-versicolor
  * Iris-virginica

## 📈 Model Performance

| Model               | Accuracy |
| ------------------- | -------- |
| Logistic Regression | 100%     |
| KNN                 | 96.67%   |
| Random Forest       | 100%     |

🏆 **Final Model: Logistic Regression**

Although Logistic Regression and Random Forest achieved the same accuracy, Logistic Regression was selected because of its simplicity, interpretability, and lower complexity.

---

## 📊 Key Insights

* Petal Length and Petal Width were the most influential features.
* The dataset is well-balanced across all three species.
* Species can be clearly separated using petal measurements.
